# Política de Privacidade - FinanceApp

**Última atualização: Abril de 2026**

## 1. Introdução

Bem-vindo ao FinanceApp. Sua privacidade é importante para nós. Esta Política de Privacidade explica como coletamos, usamos e protegemos seus dados.

## 2. Dados que Coletamos

### 2.1 Dados do Aplicativo
- **Transações Financeiras**: Valores, categorias, datas, descrições
- **Metas Financeiras**: Nome, valor alvo, progresso
- **Nome do Usuário**: Para personalizar sua experiência
- **Dados de Uso**: Como você usa o app (internamente, não enviado)

### 2.2 O que NÃO Coletamos
- ❌ Dados bancários sensíveis
- ❌ Senhas ou PINs
- ❌ Informações de geolocalização
- ❌ Dados de contato (email, telefone)
- ❌ Histórico de navegação

## 3. Como Usamos os Dados

- ✅ Para exibir seu saldo e transações
- ✅ Para gerar gráficos e insights
- ✅ Para calcular metas e economias
- ✅ Para melhorar a experiência do usuário

## 4. Armazenamento de Dados

**Todos os dados são armazenados LOCALMENTE no seu dispositivo:**
- ✅ Banco de dados SQLite local
- ✅ AsyncStorage para preferências
- ✅ Nenhum envio para servidores externos
- ✅ Você tem controle total dos seus dados

## 5. Compartilhamento de Dados

**Nunca compartilhamos seus dados com terceiros:**
- ❌ Não vendemos dados
- ❌ Não compartilhamos com publicidade
- ❌ Não sincronizamos com nuvem (a menos que você autorize em futuras versões)

## 6. Exclusão de Dados

Para deletar todos os seus dados do app:
1. Abra Configurações do iOS/Android
2. Vá para FinanceApp
3. Selecione "Deletar App"
4. Escolha "Deletar App e Dados"

**Resultado**: Todos os dados financeiros são permanentemente deletados.

## 7. Direitos do Usuário (LGPD/GDPR)

Você tem direito a:
- ✅ Acessar seus dados
- ✅ Corrigir informações imprecisas
- ✅ Deletar seus dados
- ✅ Solicitar portabilidade de dados

## 8. Segurança

Implementamos medidas para proteger seus dados:
- ✅ Armazenamento local (sem transmissão)
- ✅ Sem conexões de rede para dados financeiros
- ✅ Atualização regular de segurança
- ✅ Código aberto para transparência (GitHub)

## 9. Atualização desta Política

Esta política pode ser atualizada periodicamente. Notificaremos sobre mudanças significativas.
